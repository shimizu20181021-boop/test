# Observation of Living Organisms（ミクロ世界プロトタイプ）

## 起動方法（VS Code）
1. `Observation of Living Organisms/index.html` を開く
2. Live Server の `Go Live` で起動

## 操作
- 左下の「設定」ボタンで一時停止して、ミクロ視点は個体数/転生個体数、マクロ視点は転生上限や繁殖系の設定を変更できます
- ミクロ視点: 転生個体数（5〜100）を設定できます
- マクロ視点: マップサイズ、転生上限（生物+植物 / 群れ単位）、出産数/繁殖回数、植物の繁殖回数、食事回復量を設定できます
- 右下の「ミクロ視点 / マクロ視点」で表示を切り替えられます（切り替えてもミクロは動き続けます）
- マクロ視点は広いマップなので、ドラッグまたは `WASD` / 矢印キーでカメラ移動できます
- マクロの生物発生上限（転生数）に達すると、ミクロの合体活動は停止します（マクロの動物が0体になると再開）
